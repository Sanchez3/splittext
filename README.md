#文字动画效果

[CodePen](https://codepen.io/sanchez3/full/MOJYXq/)

主要分为4种动画效果

- Filp、Scale、Words效果主要利用css `transform3d`， `perspective`，伪3d动画效果。
- Blur效果利用css `filter` 滤镜，模糊效果。
- Shine效果利用`text-shadow` 阴影，发光效果。
- 2D CANVAS效果利用`getImageData()` 像素数据，粒子效果。
